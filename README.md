# Экспорт архивных заказов из aliexpress.ru в Excel с использованием управления браузером FireFox
Это не полноценный проект, а скрипт order_list.py и др. для экспериментов и возможной доработки

* Проверялось на python 3.10, убедитесь, что используемые библиотеки установлены
* Сначала получите кук - get_cookies.py, логинитесь (может падать!) -> cookies.pickle. Кук работает какое-то время, потом надо новый получать
* Скрипт завязан на разметку, особо опасные места помечены !!. Если не работает - возможно разметка изменилась
* Если что-то изменится в разметке или новое понадобится - см. режим разработчика в браузере и доку на selenium
* для того чтобы не банили имеется задержка sleep при запросах

## Об экспорте
* Эксорт идет в линейнуюю таблицу "по товарам"
* Количество экспортируемых заказов - сколько отражается, можно полистать - см. константу NUM_SCROLL
* Трэк, полученно/не получено не анализируется - можно доделать
* Если года в дате заказа нет, то он в коде скрипта устанавливается в 2026
* Переноса по словам в многосточных ясейках нет - можно настроить уже в excell
* Повторно заказы не экспортируются - см. saved_orders.json, то есть можно экспртировать только то, чего еще не было
